# Sound effects

Three small MP3s expected here. App still works without them (silent).

| File | Purpose |
|---|---|
| `bell.mp3` | Soft bell when wheel lands. ~2s |
| `drumroll.mp3` | Drumroll under the wheel spin. ~2s |
| `applause.mp3` | Polite applause for door-applause flash. ~3s |

**Where to source** (free, no account needed):
- https://freesound.org/ — search "temple bell single", "short drumroll cymbal", "polite applause short"
- Pixabay sounds, BBC Sound Effects, etc.

Drop the files in this directory, push, done.
